Less Autocompiler

for Windows

So you want to use the super cool less css syntax for you CSS files, but you are stuck on Windows without any help of tools like the Less App :( ?

Then this little app might help you. It offer a minimal UI to compile your less files. Point to a folder to watch, and hit Go!

This is just a minimal wrapper, you still need the less compiler working on your system. The zip file below can help you with that
